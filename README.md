# Turtle Crossing Game (Python)

## What is Turtle Crossing?

Turtle Crossing is a simple survival game where you help a turtle cross a busy road filled with moving cars.

Sounds easy… until the cars get faster and your reactions aren’t fast enough.

One wrong move and it’s game over.

---

## Features of the Code

- Player-controlled turtle movement  
- Randomly generated moving cars  
- Increasing difficulty (speed increases each level)  
- Collision detection with cars  
- Level progression system  
- Score/level display  

---

## Tech Used

- Python  
- `turtle` module for graphics  
- `random` module for car generation  

---

## How to Play

- Press the Up arrow key to move the turtle forward  
- Avoid all incoming cars  
- Reach the top of the screen to level up  

Keep going as long as you can without getting hit.
